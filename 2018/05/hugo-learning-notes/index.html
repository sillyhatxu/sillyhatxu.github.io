Hugo Learning Notes

hugo golang blog

Posted by Cookie Xu Saturday, May 26, 2018

Install

Install hugo

brew install hugo

hugo version

Create a New Site

hugo new site sillyhat-hugo

Add a Theme

cd sillyhat-hugo

cd themes

git clone https://github.com/giraffeacademy/ga-hugo-theme.git

echo 'theme = "ga-hugo-theme"' >> config.toml

Add Some Content

hugo new A.md

Start the Hugo server

hugo server -D

Navigate to your new site at http://localhost:1313/
